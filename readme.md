## Smart Allocation & Workflow Management System

### Problem Statement
Manual allocation and workflow tracking leads to errors, lack of visibility,
and inefficient operations in enterprise environments.

### Solution
A custom Salesforce application built using Lightning Web Components and Apex
to manage allocations, workflows, and role-based actions securely and at scale.

### Tech Stack
- Salesforce LWC
- Apex (Service Layer)
- SOQL
- Permission Sets
- GitHub
